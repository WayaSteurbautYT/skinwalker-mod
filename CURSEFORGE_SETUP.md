# CurseForge Publishing Guide

## Step 1: Create CurseForge Account

1. Go to https://authors.curseforge.com/
2. Click **Sign Up** or **Log In**
3. Complete account verification if needed

## Step 2: Create New Project

1. After logging in, click **Create Project**
2. Select **Minecraft Mods** category
3. Fill in project details:

### Basic Information
- **Project Name**: `Skinwalker Mod`
- **Project Slug**: `skinwalker-mod` (auto-generated, can edit)
- **Summary**: "Immersive roleplay mod with AI bots, skinwalker mechanics, and story events"
- **Project Type**: Mod
- **Game Version**: Minecraft
- **Category**: Roleplay / Adventure

## Step 3: Upload First Release

1. Click **Create New File** or **Upload File**
2. Select the JAR file: `build/libs/skinwalker-roleplay-2.0.0.jar`
3. Fill in release details:

### Release Information
- **Version**: `2.0.0`
- **Release Type**: 
  - **Alpha** - If still testing
  - **Beta** - If mostly stable but has known issues
  - **Release** - If fully tested and stable
- **Game Versions**: Select `1.20.1`
- **Loaders**: Select `Fabric`
- **Changelog**:
```
## v2.0.0 - Initial Release

### Features
- AI Bot System with multi-API support
- Complete role system with YouTuber role
- Infection system with visual bar
- Coin system and shop
- Trading system

- Win/Loss system
- Particle effects

Requires Fabric Loader 0.15.11+ and Fabric API 0.92.2+1.20.1
```

4. Click **Save** or **Submit for Review**

## Step 4: Wait for Approval

- CurseForge will review your submission
- Usually takes 1-3 business days
- You'll receive an email when approved/rejected

## Step 5: After Approval

1. Your mod will be live on CurseForge
2. Share the link: `https://www.curseforge.com/minecraft/mc-mods/skinwalker-mod`
3. Update your README with the CurseForge link
4. Consider adding a "Download on CurseForge" badge

## Tips for Success

- **Good Screenshots**: Show key features in action
- **Clear Description**: Explain what makes your mod unique
- **Regular Updates**: Keep the mod updated with new features
- **Respond to Comments**: Engage with users
- **Bug Fixes**: Fix reported issues quickly
